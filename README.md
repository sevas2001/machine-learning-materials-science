# Machine Learning aplicado a Ciencia de Materiales

Proyecto desarrollado para la asignatura *Machine Learning Aplicado a Ciencia de Materiales*.

## Objetivo
Desarrollar modelos supervisados de clasificación y regresión para predecir propiedades de materiales utilizando datos del Materials Project.

## Casos de estudio
1. Clasificación de materiales semiconductores  
2. Clasificación de materiales fotovoltaicos  
3. Predicción de estabilidad termodinámica  
4. Predicción del band gap (regresión)

## Tecnologías utilizadas
- Python
- Scikit-learn
- Pandas
- Matplotlib / Seaborn
- Materials Project API

## Resultados
Comparación de algoritmos, métricas de rendimiento (AUC, F1-score, R²) e interpretación física de los resultados.
